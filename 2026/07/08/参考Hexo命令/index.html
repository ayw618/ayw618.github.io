一、 Hexo 常用命令速查表

分类 命令 简写 作用说明
初始化 hexo init <文件夹> - 创建新的 Hexo 站点文件夹
新建文章 hexo new <标题> hexo n 在 source/_posts 下创建 .md 文件
新建页面 hexo new page <页面名> - 在 source 下创建独立页面（如 about）
本地预览 hexo server hexo s 启动本地服务，地址 http://localhost:4000
生成静态文件 hexo generate hexo g 将 Markdown 渲染为 HTML 到 public 目录
部署发布 hexo deploy hexo d 推送 public 内容到远程仓库（如 GitHub）
清除缓存 hexo clean - 删除 public 和 db.json（解决不更新问题）
组合发布 hexo clean && hexo g -d - 最常用！清理 + 生成 + 自动部署
